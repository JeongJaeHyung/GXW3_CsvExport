# compile.py

## 목적

`compile` 모듈은 GXWorks3 프로젝트의 컴파일 과정을 자동화하는 역할을 합니다. 사용자가 수동으로 컴파일 명령을 실행하는 과정을 스크립트로 대체합니다.

## 주요 기능

### `work()` 함수

- **설명**:
  이 함수는 프로젝트를 컴파일하기 위해 필요한 키 입력 시퀀스를 실행합니다. 주로 단축키를 사용하여 컴파일 관련 메뉴를 호출하고 실행합니다.

- **동작 순서**:
  1. `Just.key_press("f4")`:
     - F4 키를 누릅니다. 이는 GXWorks3에서 'Compile' 기능을 실행하는 기본 단축키 중 하나일 수 있습니다.
  2. `Just.hotkey_press(['ctrl', 'shift', 'q'])`:
     - `Ctrl + Shift + Q` 핫키를 누릅니다. 이는 컴파일과 관련된 특정 동작(예: 모든 프로그램 컴파일 또는 특정 빌드 옵션)을 트리거할 수 있습니다.
  3. `Just.key_press("enter")`:
     - 'Enter' 키를 누릅니다. 이는 컴파일 확인 대화상자나 옵션 창에서 '확인' 또는 '실행' 버튼을 누르는 것과 같은 역할을 합니다.
  4. 컴파일 완료 로그를 출력하며 함수를 종료합니다.

## 의존성

- **`module.Just`**: 즉각적인 키 입력 및 핫키 입력을 위해 사용됩니다. 이 모듈은 별도의 대기 시간 없이 신속하게 키 이벤트를 발생시켜야 할 때 사용됩니다.
