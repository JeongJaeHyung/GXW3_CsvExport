# GXWorks3 Program Exporter

## 프로젝트 개요

이 프로젝트는 GXWorks3에서 작성된 PLC 프로그램을 CSV 파일로 내보내는 자동화 도구입니다. 이 도구는 GUI를 통해 사용자로부터 프로젝트 경로와 내보내기 설정을 입력받아, GXWorks3를 자동으로 조작하여 프로그램을 내보냅니다.

## 프로젝트 구조

```
.
├── action/              # GXWorks3 자동화 작업 관련 모듈
├── UI/                  # 사용자 인터페이스 관련 모듈
├── module/              # 키보드, 마우스, 화면 등 저수준 제어 모듈
├── icons/               # UI에 사용되는 아이콘
├── docs/                # 프로젝트 문서
├── core.py              # 애플리케이션의 핵심 로직
├── main.py              # 애플리케이션 진입점
├── requirements.txt     # 프로젝트 의존성
└── README.md            # 프로젝트 개요
```

### 주요 디렉토리 설명

- **action**: GXWorks3를 열고, 트리 구조를 탐색하고, 컴파일하고, CSV로 내보내는 등의 자동화 작업을 수행하는 스크립트를 포함합니다.
- **UI**: PySide6를 사용하여 빌드된 애플리케이션의 그래픽 사용자 인터페이스(GUI) 컴포넌트를 포함합니다.
- **module**: 키보드 입력, 마우스 이동 및 클릭, 화면 캡처 및 이미지 인식과 같은 기본적인 상호 작용을 위한 모듈을 포함합니다.
- **icons**: 애플리케이션의 GUI에 사용되는 이미지 파일들을 포함합니다.
- **docs**: 프로젝트의 문서들을 포함합니다.
