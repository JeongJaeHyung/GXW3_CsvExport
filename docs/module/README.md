# Module 모듈

`module` 모듈은 키보드, 마우스, 화면과 같은 저수준의 사용자 입력을 시뮬레이션하고 화면을 분석하는 기능들을 제공합니다. 이 모듈들은 `action` 모듈에서 GXWorks3 애플리케이션을 자동화하는 데 사용됩니다.

각 파일은 특정 입력 장치나 기능에 대한 클래스와 메서드를 포함하며, 동작 방식에 따라 `Just`, `Waited`, `Found` 세 가지 클래스로 나뉩니다.

- **`Just`**: 즉시 동작을 수행합니다.
- **`Waited`**: 지정된 시간만큼 기다린 후 동작을 수행합니다.
- **`Found`**: 화면에서 특정 아이콘을 찾은 경우에만 동작을 수행합니다.

## 모듈 상세 설명

각 모듈의 상세한 기능과 동작 방식은 아래 링크에서 확인할 수 있습니다.

- **`keyboard.py`**:
  - **목적**: 키보드 입력을 시뮬레이션합니다.
  - **[상세 설명](./keyboard.md)**

- **`location.py`**:
  - **목적**: 이미지의 화면상 위치를 찾는 기능을 제공합니다.
  - **[상세 설명](./location.md)**

- **`mouse.py`**:
  - **목적**: 마우스 입력을 시뮬레이션합니다.
  - **[상세 설명](./mouse.md)**

- **`screen.py`**:
  - **목적**: 화면에서 특정 이미지를 찾고 위치를 반환합니다.
  - **[상세 설명](./screen.md)**
