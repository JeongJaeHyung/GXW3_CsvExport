# 사용 전 필수 세팅
1. GX Works3에서 모든 창을 닫는 단축키를 설정해야 합니다.
   1. GX Works3를 실행합니다.
   2. Tool을 클릭합니다.
   3. Shortcut Key를 클릭합니다.
   4. command folder를 기준으로 가장 아래에서 5번째 폴더 Window로 이동합니다.
   5. "Close All Windows"에 Key1 부분의 빈칸을 더블 클릭한 뒤 단축키 Ctrl + Shift + Q 를 눌러 등록합니다.
2. 모든 tree는 점진적으로 최 하위 객체부터 최 상위 객체까지 닫을 수 없을때 까지 닫아져야 합니다.
   1. 프로그램을 실행하기 전 GX Works3 Navigation에는 아래와 같은 요소들만 보여야 합니다.
        Project
            Module Configuration
            Program
            FB/FUN
            Lable
            Device
            Parameter
    2. 또한 실행 전 Parameter를 한번만 클릭해 노란색으로 선택되어야 합니다.
       1. 이때 Parameter를 여는것이 아닌 한번만 클릭하여 선택됨 상태로만 만드는 것 입니다.
3. GX Works3 화면은 항상 최대화 되어 있어야 합니다.
4. CSV를 저장할 폴더에 한번은 Export해줘야 경로가 설정됩니다.
